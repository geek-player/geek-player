SpringMVC：

  1. 基于SpringIOC容器运行 –> 所有的controller交给Spring创建
  2. 轻量级web mv框架，围绕DispatcherServlect，调用Servlet设计 –> Servlet可做的，SpringMVC都可以做，而且更加强大

SpringMVC开发步骤

  1. 导入spring-webmvc
  2. web.xml配置文件

web.xml

3.SpringMVC-config.xml配置文件

SpringMVC-config.xml

  1. Controller类

    • 接收数据：

      • 基本类型：变量名与传递的key一致可以自动类型转换
      • 自定义类：类中属性名与传递的key一致，就可以自动类型转换
      • 任意名称：@RequestParam("key") Object value

    • 跳转：

      • 传值方式
        • ModelAndView
        • Request
        • Session
        • Model
      • 跳转方式
        • ModelAndView
        • request.getRequestDispatcher…
        • response.sendRedirect…
          默认返回值是String，则返回值位置写的字符串就是转发到对应的jsp名字
          若想转发到controller，需要返回值位置写”forward:index.jsp”
          重定向到jsp:”redirect:xx.jsp”
          controller:”redirect:controller请求地址”

    • mapper.xml：

      • namespace: mapper类的全路径
      • id：抽象方法名
      • parameterType：形参
      • resultType/resultMap：数据返回类型
